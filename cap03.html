  3. css

    O CSS é uma linguagem de estilo usada para descrever a apresentação de documentos HTML ou XML.

    O HTML é a linguagem de marcação padrão para documentos projetados para serem exibidos em um navegador web.

    O web é o conjunto de tecnologias e protocolos que permitem a comunicação e o compartilhamento de informações na internet.
